Joomla-PDF

Joomla-PDF is an HTML5 technology experiment that explores building a faithful and efficient Portable Document Format (PDF) renderer without native code assistance.

The idea was from spawned from pdf.js

Our goal is to create a Joomla plugin that provides the same functionality based on their work.

Getting the code

To get a local copy of the current code, clone it using git:

$ git clone git://github.com/N6REJ/Joomla-pdf.git joomla-pdf
$ cd joomla-pdf

or https://github.com/N6REJ/Joomla-pdf.git

Next, you need to start a local web server the view we're currently going to start with will be at http://localhost/Joomla-pdf/web/viewer.html you may change which pdf is being rendered by changing line 6 of viewer.js
